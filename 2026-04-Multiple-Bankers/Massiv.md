# Massiv Command Reference

| Command | Description |
| :--- | :--- |
| **back** | Simulates pressing the back button. |
| **check** | Registers device with info and backup C2 address. |
| **click** | Performs a simulated click at specified screen coordinates. |
| **clipboard** | Injects text into the clipboard and focused editable fields. |
| **disableGraphics** | Disables graphics capture and stops VNC streaming. |
| **enableGraphics** | Enables graphics capture and requests MediaProjection permission. |
| **graphicKey** | Detects lock-screen pattern and automatically draws a supplied pattern. |
| **home** | Simulates pressing the Home button. |
| **inject** | Injects banking app HTML received from the server. |
| **injectPattern** | Injects pattern overlay on top of the device. |
| **injectPin** | Injects PIN overlay on top of the device. |
| **requestMiuiBackground** | Opens MIUI Auto-Start permission screen for Xiaomi devices. |
| **keypress** | Simulates keypresses (characters, backspace, enter, etc.) via Accessibility. |
| **overview** | Simulates pressing the Recent Apps button. |
| **power** | Simulates opening the Power Menu. |
| **push** | Creates a fake notification. |
| **requestAdmin** | Requests device administrator permission. |
| **swipe** | Performs a simulated swipe gesture. |
| **swipeDown** | Simulates opening the Notifications shade. |
| **swipeUp** | Programmatically opens the Quick Settings panel. |
| **wakeUp** | Wakes the device screen using PowerManager API. |
| **blackscreen** | Places an "update" overlay on top of the screen. |
| **disableBlackscreen** | Removes the update overlay. |
| **requestBattery** | Requests exemption from battery optimizations. |
| **updateInjects** | Manages HTML injection files and target app matching. |
| **renicilization** | Forces the WebSocket to reconnect. |
| **uninstallApp** | Opens app info for a specific package for manual uninstallation. |
| **requestGoogleProtect** | Opens Google Play Protect or Security settings. |
| **requestFileAccess** | Handles remote file management (listing/downloading). |
| **fileManager** | Validates storage permissions for directory and file operations. |
| **requestWriteSettings** | Requests the WRITE_SETTINGS permission. |
| **requestReadSms** | Requests SMS permission. |
| **requestPush** | Requests POST_NOTIFICATIONS permission. |
| **updateInterfaceParams** | Sets adaptive or fixed update parameters. |
| **installApk** | Downloads and triggers installation of an APK via Accessibility. |
| **requestInstallPermission** | Requests Install Package permission. |
| **reinitialization** | Force reconnects to the websocket server. |
