# SaferRat Command Reference

| Command | Description |
| :--- | :--- |
| **export_sms** | Collect list of SMS and send to C2 in a ZIP file. |
| **send_sms** | Sends an SMS message from the infected device. |
| **export_call_log** | Leaks device call history in a ZIP file. |
| **show_notification** | Create a custom notification. |
| **send_ussd** | Initiates a USSD request programmatically. |
| **VNC(commands)** | VNC session for remote control including Volume control and user actions. |
| **Volume_up / down / set** | Manipulate system audio settings. |
| **Set_touch_model** | Enable real-time or delayed touch input. |
| **Swipe_up / down / left / right** | Perform directional swipe gestures. |
| **Show_overlay_black / web / etc** | Display various malicious overlays (black, update, dimmed). |
| **request_projection** | Request MediaProjection Permission for screen capture. |
| **start_camera / microphone** | Remote audio and video capture. |
| **launch_app** | Open a specific application. |
| **Start_stream / stop_stream** | Screen sharing control. |
| **Enable_anti_delete** | Prevent uninstallation of the malware. |
| **Get_apps** | Retrieve the list of installed applications. |
| **Wake_up** | Prevent the screen from locking. |
| **Workmanager_run_now** | Monitor and recover malware components. |
| **Lock_screen / Unlock** | Remote device locking/unlocking. |
| **Uidump** | Get UI information from the current screen. |
| **Update_injection** | Inject malicious URLs to steal banking/crypto credentials. |
